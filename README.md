## Getting Started
* Clone the repo with --recurse-submodules
* Run 'Generate_VS20XX_Project.bat'
* Open the project inside the folder called 'CMakeFiles'
* Build and run

### Prerequisites
* CMake

Modify 'Generate_VS20XX_Project.bat' to get this to build on other platforms
